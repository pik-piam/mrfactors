# calcFacReq

This function calculates factor requirement costs (constant 2017
US\$MER/tDM) on regional level, based FAO databases

## Usage

``` r
calcFacReq(splitSectors = FALSE)
```

## Arguments

- splitSectors:

  if TRUE requirements for labor and capital will be reported separately

## Value

MAgPIE object
