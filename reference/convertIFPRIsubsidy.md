# convertIFPRIsubsidy

disaggregates EU subsidies for crops and livestock from IFPRI table to
country level

## Usage

``` r
convertIFPRIsubsidy(x)
```

## Arguments

- x:

  magpie object provided by the read function

## Value

magpie object of agricultural subsidies
